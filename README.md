# PowerBI_Dashboard_Project-3
Project_Name: Sales Analysis Dashboard |
Used_Tools: Power BI, Power query, Data Modeling, Dax

The objective of this project was to analyze sales performance to identify key trends, top-performing continents, countries, and regions, and opportunities for revenue growth.
#Project_Highlights:
1. This Sales Analysis Project was actually focused on Continent, Country, and Region.
2. Built dynamic dashboards with filters for products, regions, countries, and territories
3. Added new columns and measures, such as Age group, Sales Amount, and Yellow order Quantity, which were not present in the data.
4. Made relations among the tables from the data sets, which makes the dataset easier to analyze.
5. At last, I have tried to conclude the Dashboards based on the 'Decomposition Tree', where we can easily understand the actual sales amount of the Continents, Countries, and Regions.
